Function calling allows a large language model (LLM) to interact with external tools or APIs. 

Parameters `tools` and `tool_choice` of our OpenAI-compatible chat API are now accepted for models with this capacity.

Read [our dedicated documentation](https://www.khulnasoft.com/en/docs/ai-data/managed-inference/reference-content/function-calling-support/) and [tutorial to get started](https://www.khulnasoft.com/en/docs/tutorials/building-ai-application-function-calling)!
